Project Name: Elderly App
Skills: App Development using Flutter
Description:

Elderly App is an app for elderly people to remind them to take medicines on time whenever they are alone. Not only medicines details but it also keep records of all the appointments of doctor with time .The notifications are send to the user on time so to remind them of their medication details.It also allows the guardian to keep track of all the details of their respective elder people. Guardian also play a major in the app to keep their elder people healthy by updating all the medical details.The photo of doctor's prescriptions can also be uploaded in the app .Elderly App is user friendly as it is multilingual (English, Hindi). Project Name:Catch Practice Robot
Skills: Computer Vision and ROS Gazebo
Description:

The main idea of the project is to build an automatic robot which can aim at the required target once it is identified. The robot will idetect the person nearby it and will shoot the ball towards that person. This can be achieved by adjusting the shooting components to the required positions for launching the projectile. For person detection HOG descriptor is use with Support Vector Machine model(SVM). For this project we have basically used different concepts like Computer Vision(Image Processing) , mechanics projectile equation and ROS simulation. Together summing up all these concepts and applying it we could build our automatic Catch Practice Robot. View Catch Practice Robot Project
